## `~/installation`

Get up and running in seconds.

```bash
# 1. Clone the repository
git clone https://github.com/1mystic/pycrumbs.git

# 2. Navigate to project root
cd pycrumbs

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
